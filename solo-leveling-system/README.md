# Solo Leveling "System": Holistic Daily Quest & Leveling Web App

Welcome to **The System**. This workspace contains the files for the interactive, dark-mode sci-fi web application that functions as an immersive, real-world health and workout RPG tracker.

---

## 📂 Project Directory Structure

All files in this workspace are crafted using pure, optimized Vanilla Web Technologies (HTML5, CSS3, ES6 JS) to ensure zero dependency overhead and fast loading:

*   **[`index.html`](file:///C:/Users/PC/.gemini/antigravity/scratch/solo-leveling-system/index.html)**: The core layout skeleton. Houses the tabs for STATUS, DAILY QUESTS, STATS, NOURISHMENT, DUNGEONS, and INVENTORY, along with the HTML5 Centipede Survival Canvas and the Appraisal Modals.
*   **[`styles.css`](file:///C:/Users/PC/.gemini/antigravity/scratch/solo-leveling-system/styles.css)**: Implements the sleek cybernetic dark HUD theme using HSL tailored color variables, neon glow drop-shadows, glassmorphic backdrop filter styling, and responsive CSS grids.
*   **[`app.js`](file:///C:/Users/PC/.gemini/antigravity/scratch/solo-leveling-system/app.js)**: The operational engine. Runs the mathematical algorithms for physical health assessments, turn-based battle rolls, inventory-gacha, sleep and water logging, and the Penalty Quest game loop.

---

## ⚡ Active System Subsystems

1.  **Awakening Appraisal (BMI, BMR, TDEE)**: Analyzes physical height, weight, age, and activity parameters to calculate real-world indices and award starting Hunter Ranks (E-Rank to B-Rank).
2.  **Immersive Workout Tracker**: Easy (Conditioning), Medium (Vanguard), or Hard (Monarch 100/100/100/10km) daily exercise loggers.
3.  **Wellness & Meditation Logs**: Integrates circular deep-breathing timers, sleep duration inputs, and volumetric water tracking.
4.  **Attribute Point Allocation**: Freely distribute level-up points to Strength (STR), Agility (AGI), Vitality (VIT), Sense (SEN), and Intelligence (INT) to strengthen combat formulas.
5.  **Dungeon Gates (Auto-Battler RPG)**: Spend gate keys in your inventory to enter portals (E-Rank Goblin Den to S-Rank Jeju Island Ant Colony) and fight turn-based RPG matches where damage and evade chances scale directly with your stats.
6.  **Loot Gacha openeing**: Open Daily reward chests with pop animations and particles to unlock weaponry like *Kasaka's Venom Fang* or the *Monarch's Dagger*.
7.  **Penalty Quest Survival canvas**: Survive dodging incoming desert sand centipedes on an HTML5 canvas if you fail to complete your daily goals in time.

---

## 🚀 How to Launch & Develop

1.  **Run locally**: Double-click or open **`index.html`** in any web browser to run the app immediately (no web servers required!).
2.  **Modify and Code**: As this directory is now established as your active coding workspace, all new features, styling extensions, or dungeon combat enhancements can be written directly to `styles.css` and `app.js`.
3.  **Persistence**: The state manager automatically backs up all Level, EXP, Stats, and Inventory modifications to browser `localStorage` on any change. To wipe progress and start as an E-Rank hunter again, use your browser console to run `localStorage.clear()` or hit the Recalibrate button.
